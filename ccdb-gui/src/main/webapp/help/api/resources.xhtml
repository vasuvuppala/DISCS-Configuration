Resources

A resource is an entity that is managed by CCDB. For example, a device, a slot, a beampath, a device type, an approval status etc. The #{msgs.title} API provides access to the resources or entities that CCDB manages. These entities and their attributes are described below. Note that the names of attributes are important as they are used in the various representations of these resources (see Resource Representations Section below).
